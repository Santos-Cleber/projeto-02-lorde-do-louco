# Projeto 02 — Lorde do Louco

Página temática do personagem **O Louco** do anime *Lord of Mysteries*, desenvolvida como segundo projeto do portfólio pessoal.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts (Playfair Display e DM Sans)

## Funcionalidades

- Efeito de digitação na frase mística com `setInterval`
- Revelação progressiva dos poderes com `setTimeout`
- Layout responsivo com estilização temática dark

## Como visualizar

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador

## Aprendizados

- Manipulação do DOM com `getElementById` e `getElementsByClassName`
- Uso de `setInterval` e `setTimeout` para criar efeitos temporizados
- Loop `for` para percorrer coleções de elementos HTML
- Alteração de estilos via `element.style` no JavaScript
